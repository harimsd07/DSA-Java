Java's Big Picture: JDK vs. JRE

JDK = JRE + Development Tools (Java Development Kit)
JRE = JVM + Library Classes (Java Runtime Environment)
JVM (Java Virtual Machine)
JIT (Just-in-Time)
  • JDK (Java Development Kit)

    You need this to write and compile Java code. It includes the JRE plus development tools (like the javac compiler).

  • JRE (Java Runtime Environment)

    You need this to run Java applications. It includes the JVM plus all the built-in library classes (like String, System, etc.).

  • JVM (Java Virtual Machine)

    The "engine" or "factory" that actually executes the .class bytecode. This is what we animated above!

  • JIT (Just-in-Time) Compiler

    A component of the JVM that boosts performance by compiling frequently used bytecode into native machine code at runtime.
